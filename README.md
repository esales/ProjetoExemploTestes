# Projeto Exemplo de Testes

Aplicação CRUD desenvolvida com:

- Node.js
- Express
- Sequelize
- SQLite
- Handlebars

# Instalação

## Clonar o projeto

```bash
git clone https://github.com/esales/ProjetoExemploTestes.git
```

## Entrar na pasta

```bash
cd ProjetoExemploTestes
```

## Instalar dependências

```bash
npm install
```

## Executar aplicação

```bash
npm start
```

## Acessar aplicação

No navegador, acessar http://localhost:3000
